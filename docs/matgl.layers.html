class matgl.layers._activations.SoftExponential(alpha: float | None = None)¶

Bases: Module

Soft exponential activation. When x < 0, SoftExponential(x,alpha) = -log(1-alpha(x+alpha))/alpha When x = 0, SoftExponential(x,alpha) = 0 When x > 0, SoftExponential(x,alpha) = (exp(alpha*x)-1)/alpha + alpha.

References:

  • See related paper:

https://arxiv.org/pdf/1602.01321.pdf
